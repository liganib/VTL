/resolve GET Method

Returns the value of a given object defined in a given session.

HTTP Request

Request fields
Parameter Description

uuid

The unique identifier of the VTL session

alias

The alias of the object whose value is returned
Request cookies
Name Description

JSESSIONID

The Java Servlet server session cookie.

HTTP Response

Response fields
Path Type Description

type

String

Always SCALAR for scalar values

domain

String

The VTL domain of the scalar value

value

Number

The computed scalar value
Sample response body
{"type":"SCALAR","domain":"integer","value":3}

Invocation examples

Curl command
$ curl 'http://localhost:40118/resolve?alias=c&uuid=9544e822-c806-4b69-80c5-4e42bfe2cf36' -i -X GET \
    -H 'Accept: application/json' \
    --cookie 'JSESSIONID=A22ED1DEA9E93C561C30537B5D91DCF5'
http command
$ http GET 'http://localhost:40118/resolve?alias=c&uuid=9544e822-c806-4b69-80c5-4e42bfe2cf36' \
    'Accept:application/json' \
    'Cookie:JSESSIONID=A22ED1DEA9E93C561C30537B5D91DCF5'
